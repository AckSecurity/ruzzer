# Ruzzer

This tool was created to inject payload on the software input.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'Ruzzer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install Ruzzer

## Usage

ruzzer.rb [OPTIONS]
